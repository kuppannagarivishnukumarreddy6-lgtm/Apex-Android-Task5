# Project Overview

The Student Registration App is an Android application developed using Kotlin and Jetpack Compose. The application enables users to log in, register student information, navigate between screens, and perform API operations using Retrofit.

The project demonstrates Android UI development, navigation, REST API integration, testing, and version control using GitHub.

---

# Features Implemented

## User Authentication

* Login Screen
* Username and Password Validation
* Invalid Login Handling

## Student Registration

* Name Input
* Email Input
* Phone Number Input
* Form Validation
* Registration Success Message

## Navigation

* Login Screen to Registration Screen
* Registration Screen to Success Screen
* Back Navigation Support

## API Integration

* GET Request
* POST Request
* PUT Request
* DELETE Request

## User Interface

* Built using Jetpack Compose
* Material Design Components
* Responsive Layout
